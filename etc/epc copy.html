플랜트 전기분야 설계

서비스 개요

발전소, 산업플랜트의 전기시스템 설계부터 시공까지 전 과정을 책임지는 통합 솔루션을 제공합니다.

주요 서비스

전력계통 설계
  • 송배전 시스템 설계
  • 보호계전 시스템 설계
  • 접지시스템 설계
배전반 설계
  • MCC(Motor Control Center)
  • PCC(Power Control Center)
  • 배전반 Layout 설계
케이블 시스템
  • 케이블 선정 및 경로 설계
  • Cable Tray 설계
  • Cable Schedule 작성
조명 시스템
  • 실내외 조명 설계
  • 비상조명 시스템
  • 경관조명 설계

플랜트 계측분야 설계

서비스 개요

플랜트 운영에 필수적인 계측제어 시스템의 설계와 구축을 통해 안전하고 효율적인 플랜트 운영을 지원합니다.

주요 서비스

제어시스템 설계
  • DCS(Distributed Control System)
  • PLC(Programmable Logic Controller)
  • SCADA 시스템
계측기기 설계
  • 온도, 압력, 유량 계측
  • 레벨, 밀도 계측
  • 가스 분석 계측
안전시스템
  • SIS(Safety Instrumented System)
  • 화재감지 시스템
  • 가스감지 시스템
네트워크 시스템
  • 산업용 이더넷
  • 필드버스 시스템
  • 무선통신 시스템

전력계통 컨설팅

서비스 개요

전력계통 분야 40여년의 경험과 전문성을 바탕으로 최적의 전력시스템 솔루션을 제안합니다.

주요 서비스

계통 해석
  • 조류계산(Load Flow)
  • 단락계산(Short Circuit)
  • 안정도 해석
보호계전
  • 보호계전 협조
  • Setting 계산
  • Arc Flash 해석
전력품질
  • 고조파 해석
  • 전압변동 해석
  • Filter 설계
신재생에너지
  • 태양광 시스템 설계
  • 풍력 시스템 설계
  • ESS 시스템 설계

CGID (Computer Generated Intelligent Drawings)

서비스 개요

AI 기반 설계 자동화 시스템으로 설계 효율성을 극대화하고 오류를 최소화하는 혁신적인 설계 솔루션입니다.

주요 특징

자동화 설계
  • 도면 자동 생성
  • 설계 검증 자동화
  • 표준화된 심볼 적용
데이터 연동
  • 설계 데이터베이스 연동
  • 실시간 설계 변경 반영
  • 다양한 CAD 포맷 지원
품질 향상
  • 설계 오류 사전 방지
  • 설계 표준 준수
  • 일관성 있는 도면 품질
효율성 증대
  • 설계 시간 단축
  • 반복 작업 최소화
  • 설계 변경 관리 용이
